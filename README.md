### Info sobre los astros:
* **Círculos rojos:** Estrellas rojas.
* **Sin aura:** Estrella normal.
* **Aura púrpura y centro negro:** Agujero negro.
* **Aura cyan pequeño:** Estrella de neutrones.
* **Círculo cyan con líneas girando:** Estrella de Púlsar.

### Controles del simulador:
* `W`: Aumentar población.
* `S`: Disminuir población.
* `A`: Añadir agujero negro.
* `D`: Añadir estrella de neutrones.
* `Q / E`: Aumentar/Disminuir zoom.
* `Flechas`: Mover cámara.has del teclado: mover la camara
